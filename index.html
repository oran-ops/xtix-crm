🔗 HubSpot API — Push ישיר

🔑 שלב 1 — צור Private App Token (פעם אחת)

HubSpot דורש Private App Token עם הרשאות CRM. כך יוצרים:

1
ב-HubSpot: Settings (⚙️) → Account Setup → Integrations → Private Apps
2
לחץ "Create a private app" → שם: XTIX CRM Sync
3
לשונית Scopes — הפעל את ארבעת ה-scopes הבאים:
crm.objects.contacts.write crm.objects.contacts.read crm.objects.companies.write crm.objects.deals.write
4
לחץ "Create app" → העתק את ה-Token (מתחיל ב-pat-)
